Projects

This is a Sokoban game solver, providing an automated way to navigate and solve puzzles inherent in the Sokoban game, where the player must push crates onto designated goal areas on a board. The primary entry point of the application, the Driver class, handles command-line arguments to load specific game maps and sets the mode of play, which can either be free play or an automated bot mode. The FileReader class facilitates the reading of these maps from external files, storing them in a MapData structure that represents the game's board.

Within the core of the application, the SokoBot class encapsulates the logic required to solve the Sokoban puzzles. It analyzes the game board to identify walls, goals, the player's position, and crates, applying algorithms to determine viable moves and assess the game state for completion or deadlock conditions. Notably, it uses a Greedy Best-First Search (GBFS) algorithm, prioritizing moves that appear to lead most directly toward a solution based on a heuristic of distance. This solver dynamically adapts to the game's changing state, ensuring efficient puzzle resolution and offering a robust tool for players and developers interested in automated game solving techniques.
